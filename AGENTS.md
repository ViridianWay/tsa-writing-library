# TSA Writing Library Agent Guide

## Project purpose
This repository hosts the **TSA Writing Library**, a public-facing digital library built with Astro + Starlight. It organizes library content into clear sections for authors, topics, publications, and project background information.

## Local development
Run these commands from the repository root:

1. Install dependencies:
   - `npm install`
2. Start the development server:
   - `npm run dev`
3. Build for production:
   - `npm run build`

## Where content lives
All site content is stored under:

- `src/content/docs/`

Primary sections in this project:

- `src/content/docs/authors/`
- `src/content/docs/topics/`
- `src/content/docs/publications/`
- `src/content/docs/about/`
